BÀI 3 – GIỚI THIỆU VỀ JAVA

Mục tiêu học tập

  • Hiểu tổng quan về ngôn ngữ lập trình Java và triết lý thiết kế “Write Once, Run Anywhere”.
  • Nắm được cấu trúc cơ bản của chương trình Java, vai trò của JVM, JDK và JRE.
  • Phân biệt các loại ứng dụng Java: console, web, mobile, và enterprise.
  • Hiểu và thực hành với cấu trúc điều khiển, mảng, lớp wrapper trong Java.

1. Giới thiệu về Java

Java là ngôn ngữ lập trình hướng đối tượng, do Sun Microsystems phát triển (nay thuộc Oracle). Mục tiêu: tạo ra chương trình có thể chạy trên nhiều nền tảng nhờ cơ chế biên dịch bytecode và máy ảo JVM (Java Virtual Machine).

  • Portable – chạy trên mọi hệ điều hành có JVM.
  • Secure – có sandbox và bộ quản lý bộ nhớ tự động.
  • Robust – cơ chế xử lý lỗi mạnh mẽ, garbage collection tự động.
  • Multithreaded – hỗ trợ xử lý đa luồng.

Châm ngôn của Java: “Write Once, Run Anywhere”.

2. Môi trường phát triển Java

  • JDK (Java Development Kit): bộ công cụ dành cho lập trình viên, bao gồm compiler (javac), JVM, thư viện chuẩn, và công cụ build (jar, javadoc,…).
  • JRE (Java Runtime Environment): môi trường thực thi Java (JVM + thư viện chuẩn) dùng để chạy ứng dụng.
  • JVM (Java Virtual Machine): biên dịch bytecode sang mã máy tương ứng của hệ điều hành.

Sơ đồ:

Source Code (.java) → Compiler → Bytecode (.class) → JVM → Machine Code

3. Cấu trúc chương trình Java cơ bản

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, Java!"); } }
  • public class HelloWorld: định nghĩa lớp công khai tên HelloWorld.
  • public static void main(String[] args): điểm bắt đầu của chương trình.
  • System.out.println(): phương thức in ra màn hình.

4. Cấu trúc điều khiển trong Java

  • Câu lệnh rẽ nhánh: if...else, switch...case.
  • Câu lệnh lặp: for, while, do...while.
  • Câu lệnh nhảy: break, continue, return.
for (int i = 1; i <= 5; i++) { System.out.println("Count: " + i); }

5. Làm việc với mảng

Mảng (Array) trong Java là tập hợp các phần tử cùng kiểu dữ liệu, có chỉ số bắt đầu từ 0.

int[] numbers = {1, 2, 3, 4, 5}; System.out.println(numbers[2]); // in ra 3
  • Mảng 1 chiều: lưu trữ dãy dữ liệu tuyến tính.
  • Mảng 2 chiều: giống ma trận.
int[][] matrix = new int[2][3]; matrix[0][1] = 5;

6. Lớp Wrapper trong Java

Java hỗ trợ lớp bao (wrapper) cho các kiểu nguyên thủy để làm việc với đối tượng: Integer, Double, Boolean, Character.

int a = 10; Integer obj = Integer.valueOf(a); int b = obj.intValue();
  • Tự động boxing/unboxing: Integer x = 5; tự chuyển sang đối tượng Integer.

7. Các loại ứng dụng Java

  • Java SE: ứng dụng console hoặc desktop.
  • Java EE / Jakarta EE: ứng dụng web và doanh nghiệp.
  • Java ME: ứng dụng di động (Android dựa trên JVM riêng).

8. Ví dụ tổng hợp

class Student { String name; int age; void display(){ System.out.println(name + " - " + age); } public static void main(String[] args){ Student s = new Student(); s.name = "An"; s.age = 20; s.display(); } }
